# 准备

如果没有安装 [cnpm](`http://npm.taobao.org/`), 先使用如下方式安装 `cnpm`。

```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

使用 `git clone` 检出代码到本机

```
git clone

```


使用 `cnpm` 安装依赖

全局安装 `gulp` 用于启动gulp 构建

cd ...

```
cnpm install -g gulp
cnpm install

```

# 运行

```
gulp

```
